MY WORK

RecipEase

RecipEase is a responsive web app for recipes. It will allow people with a busy lifestyle to quickly find a recipe that uses ingredients they already have at home.

I used Adobe XD for the design, Miro for the user flows, and Prott for paper prototyping.

Let's Plan It!

Let's Plan It! is a native mobile app created for both iOS and Android. This app adheres to both Human Interface Guidelines and Material Design throughout the entire app. Let's Plan It! allows its users to plan events of all types and helps take the stress out of planning parties! It is accessible for both party-goers and planners to maximize communication amongst the guests.

I used Adobe XD for the design, Miro for the user flows, and Prott prototyping.

Pocket That

Pocket That is a money saving mobile app. It is essentially like having a financial advisor in the users pocket. This app allows the user to input their personal financial data to visualize where their money is going and gives them tools to help save up money for big events coming up.

I used Adobe XD for the design and Miro for the user flows

Task Master

Task Master is a productivity mobile app. It allows users to stay on top of their to-do lists while also assigning other users to tasks. There is also a focus mode feature which allows users to block out all other phone notifications to allow them to focus completely on one task.

I used Adobe XD for the design.
